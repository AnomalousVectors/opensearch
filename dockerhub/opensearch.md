# Anomalous Vectors OpenSearch

Custom OpenSearch image to support agentic pentest research. Stand up a local instance from the companion Git repo, point tools at the API, and use the structured data for manual, automated, or agentic analysis.

Pair with [`anomalousvectors/opensearch-dashboards`](https://hub.docker.com/r/anomalousvectors/opensearch-dashboards).
